
# Bar-Graph-Javascript

## Steps to Run the application:

     ### Step 1: Install the dependencies
        
        > npm install
        > bower install
        
     ### Step 2: Run the application
        
        > grunt serve:bar --env=apimock
        
     ### Step 3: To run the tests
        
        > grunt test
